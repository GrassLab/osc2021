## Simple Shell
| command   | description                   | 
| ----------| ----------------------------- | 
| hello     | Print hello world.            |
| help      | List commands.                |
| reset     | Reset raspi3.                 |
| loadimg   | Load new kernel.              |
| ls        | List rootfs.                  |
| cat       | Show content of file.         |
| dtb_ls    | List device tree.             |
| dtb_cat   | Parse device tree.            |
